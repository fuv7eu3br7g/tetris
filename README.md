Краткое описание проекта и цели.
	Цель - реализация игры Тетрис на языке программирования Python.

Архитектура/структура проекта:

курсовая работа/
	tetris.py         #основной файл
	Dockerfile        #рабочий Dockerfile, собирающий образ и запускающий приложение без ручных донастроек
	README.md         #документация
	.gitignore	      #исключения для Git

Запуск локально(команды) и в Docker(build/run примеры)

для запуска необходимо установить библиотеку pygame через консоль с помощью консольной команды:
	pip install pygame
после установки библиотеки игра Тетрис готова к запуску

для сборки образа в Docker используется команда:
	docker build -t tetris-game .
для запуска используетс команда:
	docker run -it --rm tetris

Управление в игре:
	стрелка влево - перемещение фигуры на 1 блок влево
	стрелка вправо - перемещение фигуры на 1 блок вправо
	стрелка вниз - перемещение фигуры на 1 блок вниз
	q - поворот фигуры на 90 градусов влево
	e - поворот фигуры на 90 градусов вправо
